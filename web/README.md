# Brujita Buena Web

Esta web recrea la experiencia de Brujita Buena con una estética más cercana a la app original de iOS, empezando por perfiles, males y la guía de Sabrina.

## Documentación

- [BIBLIA-BRUJITA-BUENA.md](../BIBLIA-BRUJITA-BUENA.md)

## Desarrollo local

```bash
cd web
npm install
npm run dev
```

Abrí http://localhost:3000 para ver la app.
